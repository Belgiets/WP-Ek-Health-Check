Check if site and database is up and return wordpress version

== Description ==

You can use url <your domain name>/healthcheck to check if your site is up and get current wordpress version of your site

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
